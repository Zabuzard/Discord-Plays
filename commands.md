# Commands

## Key 
| Symbol      | Meaning                        |
|-------------|--------------------------------|
| [Argument]  | Argument is not required.      |

## Utility
| Commands | Arguments | Description                |
|----------|-----------|----------------------------|
| Help     | [Command] | Display a help menu.       |
| info     |           | Bot info for Discord Plays |

## game
| Commands   | Arguments | Description      |
|------------|-----------|------------------|
| game-start |           | Start a new game |
